# Footer — Visual Rules

**DECISION-MAKING REFERENCE ONLY. Does not render anything.**

---

## Surface

- Background: deepest charcoal `hsl(220 18% 4%)` — one step darker
  than the page background to feel like a base plate.
- Top border: 1px `hsl(45 15% 90% / 0.06)`.
- Padding: 96px top, 48px bottom desktop. 64px / 32px mobile.

## Grid

- 4 columns desktop (1fr each), 64px column gap.
- 2 columns tablet.
- 1 column mobile, 40px row gap.

## Typography

- Column headers: 12px uppercase, letter-spacing 0.12em, weight 500,
  color `hsl(40 55% 70%)` (champagne).
- Links: 14px, weight 400, color `hsl(45 10% 70%)` resting →
  `hsl(45 25% 96%)` hover. No underline at rest, 1px underline on hover.
- Body lines (positioning, founder): 14px, color `hsl(45 10% 60%)`.

## Bottom bar

- Border-top: 1px `hsl(45 15% 90% / 0.06)`, 32px top padding.
- 12px text, color `hsl(45 10% 50%)`.
- Final CTA text link: champagne `hsl(40 55% 70%)`, no underline,
  hover adds 1px underline.

## What's intentionally missing

- No logo lockup with social row.
- No oversized brand mark filling the bottom of the page.
- No gradient divider.
- No "newsletter" input with placeholder text.

The footer should read like the end of a financial memo, not the end
of a marketing landing page.
